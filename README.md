# Advisor Service UI
This service aggregates the functionalities of the [Healthprofile Storage Service](https://github.com/introsde-mmascotti/healthprofile_storage_service), 
the [Recipe Service](https://github.com/introsde-mmascotti/recipe_service)
and the [Advisor Service](https://github.com/introsde-mmascotti/advisor_service) in a single webservice 
in order that they can be consumed by an user agent without calling different webservices.
